📘 Chapter Summary: Analysis of Financial Statements

🔹 Meaning & Objectives: Financial Statement Analysis is the process of evaluating a company's financial performance & position. Objectives: help in planning, investment decisions, assessing efficiency, and providing financial information to stakeholders.

📌 Limitations: Qualitative info ignored, historical cost, accounting conventions, personal judgment, window dressing, static nature.

🔹 Tools for Analysis: 1. Comparative Statement (absolute & % change), 2. Common Size Statement (percentage to common base), 3. Cash Flow Statement (operating, investing, financing activities).

🔹 Ratio Analysis Classification:
A) Balance Sheet Ratios: Current Ratio (2:1 ideal), Liquid/Quick Ratio (1:1 ideal).
B) Income Statement Ratios: Gross Profit Ratio, Operating Profit Ratio, Net Profit Ratio, Operating Ratio.
C) Combined/Mixed Ratios: Return on Investment (ROI), Return on Capital Employed (ROCE).

⭐ Important Formulas: Current Ratio = CA/CL, Liquid Ratio = Quick Assets/CL, Gross Profit Ratio = GP/Net Sales ×100, Operating Ratio = (COGS+Op Exp)/Net Sales ×100, Net Profit Ratio = NP/Net Sales ×100, ROCE = NP before Interest & Tax / Capital Employed ×100.

Ratio Calculation Examples (Numericals)

1) Current Assets = 1,00,000, Current Liabilities = 60,000. Find Current Ratio.
Current Ratio = 1,00,000 / 60,000 = 5:3
2) Working Capital = 50,000, Current Assets = 60,000, Stock = 10,000, Prepaid = 4,000. Find Quick Ratio.
Quick Assets = 60,000 - 10,000 - 4,000 = 46,000; Quick Liabilities = 60,000-50,000=10,000; Quick Ratio = 46,000/10,000 = 4.6:1
3) Sales = 5,40,000, Sales Return = 40,000, Purchases = 3,00,000, Opening Stock = 90,000, Closing Stock = 50,000. Find Gross Profit Ratio.
Net Sales = 5,00,000; COGS = 90,000+3,00,000-50,000=3,40,000; GP = 1,60,000; GP Ratio = 1,60,000/5,00,000×100 = 32%
4) Sales = 7,60,000, COGS = 5,20,000, Indirect Expenses = 1,20,000. Find Net Profit Ratio.
Gross Profit = 2,40,000; Net Profit = 2,40,000-1,20,000=1,20,000; NP Ratio = 1,20,000/7,60,000×100 = 15.79%
5) COGS = 7,00,000, Operating Expenses = 60,000, Sales = 10,00,000, Sales Return = 60,000. Find Operating Ratio.
Net Sales = 9,40,000; Operating Ratio = (7,00,000+60,000)/9,40,000×100 = 80.85%
